# ⚡ النشر السريع في 5 دقائق

## الخطوة 1: إنشاء GitHub Repository (2 دقيقة)

```
1. اذهب إلى https://github.com/new
2. اسم المشروع: minecraft-3d-web
3. اختر Public
4. اضغط Create
```

## الخطوة 2: رفع الكود (2 دقيقة)

انسخ الأوامر التالية في PowerShell:

```powershell
cd "c:\Users\hm430\Downloads\mop"

git init

git add .

git commit -m "Initial commit: Minecraft 3D Web Game"

git branch -M main

git remote add origin https://github.com/YOUR_USERNAME/minecraft-3d-web.git

git push -u origin main
```

**استبدل YOUR_USERNAME باسم مستخدمك على GitHub**

## الخطوة 3: نشر على Vercel (1 دقيقة)

```
1. اذهب إلى https://vercel.com/new
2. اضغط Import Git Repository
3. اختر minecraft-3d-web
4. اضغط Deploy
5. انتظر النتيجة
```

## الخطوة 4: احصل على رابط اللعبة

```
بعد النشر، سيظهر لك رابط مثل:
https://minecraft-3d-web.vercel.app

هذا هو رابط لعبتك!
```
