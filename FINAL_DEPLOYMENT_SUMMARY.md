## 🚀 خطوات النشر النهائية:

### الخطوة 1: إعداد Git (5 دقائق)

```powershell
# افتح PowerShell في مجلد المشروع
cd "c:\Users\hm430\Downloads\mop"

# هيّئ Git
git init
git add .
git commit -m "Initial commit: Minecraft 3D Web Game"
git branch -M main
```

### الخطوة 2: إنشاء GitHub Repository (2 دقيقة)

```
1. اذهب إلى https://github.com/new
2. اسم: minecraft-3d-web
3. اختر Public
4. انسخ الأوامر من GitHub
```

### الخطوة 3: رفع الكود (3 دقائق)

```powershell
# استبدل YOUR_USERNAME باسمك
git remote add origin https://github.com/YOUR_USERNAME/minecraft-3d-web.git
git push -u origin main
```

### الخطوة 4: النشر على Vercel (5 دقائق)

```
1. اذهب إلى https://vercel.com/new
2. استيراد Repository
3. اختر minecraft-3d-web
4. اضغط Deploy
5. احصل على الرابط النهائي
```
