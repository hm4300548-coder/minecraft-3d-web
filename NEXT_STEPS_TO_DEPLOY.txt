### الخطوة 1: إنشاء Repository على GitHub

أ) اذهب إلى: https://github.com/new
ب) ملء البيانات:
   - Repository name: minecraft-3d-web
   - Description: Complete 3D Minecraft-like game with infinite terrain
   - Public (مهم!)
ج) اضغط "Create repository"

═══════════════════════════════════════════════════════════════════════════════

### الخطوة 2: رفع الكود إلى GitHub

انسخ هذه الأوامر واحدة واحدة في PowerShell:

```powershell
cd "c:\Users\hm430\Downloads\mop"

git branch -M main

git remote add origin https://github.com/YOUR_USERNAME/minecraft-3d-web.git

git push -u origin main
```

⚠️ استبدل YOUR_USERNAME باسم مستخدمك على GitHub!

═══════════════════════════════════════════════════════════════════════════════

### الخطوة 3: النشر على Vercel

أ) اذهب إلى: https://vercel.com/new
ب) اضغط "Import Git Repository"
ج) اختر "minecraft-3d-web" من القائمة
د) اضغط "Deploy"
ه) انتظر انتهاء النشر (5-10 دقائق)
